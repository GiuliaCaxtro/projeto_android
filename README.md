Este projeto é um site desenvolvido com HTML5 e CSS3, com o objetivo de apresentar curiosidades sobre o sistema operacional Android, incluindo a história do seu mascote.

O layout foi pensado para ser limpo, organizado e responsivo, com uso de fontes personalizadas, cores harmônicas e boa experiência de leitura.
